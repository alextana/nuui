## Theme Customization

The Text component supports comprehensive theme customization through the `theme` prop. You can override any part of the text styling by providing a custom theme object.

### Theme Structure

The theme object is organized into sections:

- **`base`**: Base text styling classes
- **`variants`**: Typography variant classes (displayLg, headingMd, bodyMd, etc.)
- **`tones`**: Color and tone classes (default, muted, success, etc.)
- **`alignments`**: Text alignment classes (left, center, right, justify)
- **`weights`**: Font weight classes (normal, medium, semibold, bold)

### Custom Theme Example
<Canvas of={TextStories.CustomTheme} source={{ code: `<Text 
  variant="headingLg"
  :theme="{
    root: {
      variants: {
        headingLg: 'text-2xl font-black text-purple-600'
      }
    }
  }"
>
  Custom Theme Text
</Text>` }} />

### TypeScript Support

```typescript
import type { TextTheme } from './theme'

const customTheme: TextTheme = {
  base: 'font-mono tracking-wider',
  variants: {
    bodyMd: 'text-lg leading-relaxed'
  },
  tones: {
    accent: 'text-emerald-500 font-semibold'
  }
}
```
