## Theme Customization

The Button component supports comprehensive theme customization through the `theme` prop. You can override any part of the button's styling by providing a custom theme object.

### Theme Structure

The theme object is organized into sections:

- **`root`**: Main button element styling (base, variants, sizes, rounded)
- **`icon`**: Icon-specific styling (base classes, sizes)
- **`content`**: Content wrapper styling
- **`loading`**: Loading spinner styling

### TypeScript Support

```typescript

const customTheme: ButtonTheme = {
  root: {
    variants: {
      primary: 'bg-gradient-to-r from-blue-500 to-purple-600 text-white'
    }
  },
  icon: {
    base: 'drop-shadow-lg'
  }
}
```
