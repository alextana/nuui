# Tabs

A flexible and accessible tabs component system that supports various styles, layouts, and usage patterns. The tabs component provides both a simple unified interface and individual components for maximum flexibility.

## Features

- **Multiple Variants**: Default, underline, pills, and cards styles
- **Flexible Sizing**: Small, medium, and large sizes
- **Orientation Support**: Horizontal and vertical layouts
- **Animation Control**: Enable or disable animations for performance or accessibility
- **Keyboard Navigation**: Full keyboard support with arrow keys, Home, and End
- **Accessibility**: ARIA compliant with proper focus management
- **Lazy Loading**: Optional lazy loading of tab panels
- **Controlled/Uncontrolled**: Support for both controlled and uncontrolled usage
- **Custom Styling**: Extensive customization options
- **TypeScript**: Full TypeScript support with proper type definitions

## Keyboard Navigation

The Tabs component supports full keyboard navigation:

- **Arrow Keys**: Navigate between tabs
- **Home**: Jump to first tab
- **End**: Jump to last tab
- **Enter/Space**: Activate focused tab
- **Tab**: Move focus to tab panels

## Accessibility

- Full ARIA support with proper roles and attributes
- Keyboard navigation following WAI-ARIA guidelines
- Focus management and visual focus indicators
- Screen reader friendly with proper labeling
- Supports `prefers-reduced-motion` for animations
